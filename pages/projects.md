A collection of tools, scripts, and utilities designed to optimize workflows and solve specific technical bottlenecks. Most of these are born out of the necessity to automate the mundane.

### Active Utilities
*   **[[PR Title Copier]]** – A TamperMonkey utility designed for one-click extraction of GitHub Pull Request titles.
*   **[[Quick Reviews Requested]]** – A Firefox extension that provides a view of all Pull Requests currently awaiting your review.

### Terminal Snippets (Windows .bat)
Small-batch automation scripts for local environment management.
*   **[[Docker Stop All Containers]]** - Use this when the system is struggling under the weight of orphaned containers.
*   **[[Docker Delete All Containers]]** - Total wipe of all containers. Use when RAM usage becomes a threat to stability.
