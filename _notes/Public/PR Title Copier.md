**[PR Title Copier](https://greasyfork.org/en/scripts/566845-github-pr-title-copier-improved)**

Adds a "Copy" button next to PR title on GitHub (works on navigation)

## Features
- One-Click Copy: Adds a "Copy" button next to PR titles for fast, easy access.
- Always Available: Works seamlessly with GitHub's dynamic navigation—no page refreshes required.
- Visual Feedback: The button changes to a checkmark instantly to let you know the text was saved to your clipboard.

## How to use
1. Install: Copy and paste the script into a new Tampermonkey dashboard entry.
2. Navigate: Go to any GitHub Pull Request.
3. Copy: Click the Copy button next to the title; it will briefly turn into a checkmark to confirm the title is now in your clipboard
